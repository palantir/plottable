public updateExtent(rendererID: number, attr: string, extent: Array<any>): Scale

  • Defined in core/scale.ts:133

When a renderer determines that the extent of a projector has changed, it will call this function. This function should ensure that the scale has a domain at least large enough to include extent.

Parameters

  • rendererID: number

    A unique indentifier of the renderer sending the new extent.

  • attr: string

    The attribute being projected, e.g. "x", "y0", "r"

  • extent: Array<any>

    The new extent to be included in the scale.
