public _applyAnimatedAttributes(selection: any, animatorKey: string, attrToProjector: IAttributeToProjector): any

  • Defined in core/plot.ts:213

Apply attributes to the selection.

If animation is enabled and a valid animator's key is specified, the attributes are applied with the animator. Otherwise, they are applied immediately to the selection.

The animation will not animate during auto-resize renders.

Parameters

  • selection: any

    The selection of elements to update.

  • animatorKey: string

    The key for the animator.

  • attrToProjector: IAttributeToProjector

    The set of attributes to set on the selection.

Returns

any

The resulting selection (potentially after the transition)
