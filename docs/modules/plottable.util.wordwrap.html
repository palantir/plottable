isTokenizedTogether(text: string, nextChar: string, lastChar: string): boolean

  • Defined in utils/wordWrapUtils.ts:182

Given a token (ie a string of characters that are similar and shouldn't be broken up) and a character, determine whether that character should be added to the token. Groups of characters that don't match the space or line break regex are always tokenzied together. Spaces are always tokenized together. Line break characters are almost always split into their own token, except that two subsequent identical line break characters are put into the same token. For isTokenizedTogether(":", ",") == False but isTokenizedTogether("::") == True.

Parameters

  • text: string
  • nextChar: string
  • lastChar: string

Returns

boolean
