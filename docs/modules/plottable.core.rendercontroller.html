Module RenderController

The RenderController is responsible for enqueueing and synchronizing layout and render calls for Plottable components.

Layouts and renders occur inside an animation callback (window.requestAnimationFrame if available).

If you require immediate rendering, call RenderController.flush() to perform enqueued layout and rendering serially.
