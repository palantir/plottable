Module ResizeBroadcaster

The ResizeBroadcaster will broadcast a notification to any registered components when the window is resized.

The broadcaster and single event listener are lazily constructed.

Upon resize, the _resized flag will be set to true until after the next flush of the RenderController. This is used, for example, to disable animations during resize.
